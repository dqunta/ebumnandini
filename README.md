# Ebumnandini Lounge Website

A modern, immersive single-page website for **Ebumnandini Lounge** (catering, decor, and baking), built to showcase the brand's style, creativity, and attention to detail.

## Files included

- `index.html` - Main landing page
- `theme.css` - Active styling used by the live page
- `style.css` - Older alternate stylesheet kept in the project
- `main.js` - Mobile navigation, section reveal, and contact form behavior
- `*.jpg` - Assets used for the portfolio and hero sections

## How to preview

1. Open `index.html` in your browser.
2. For a local dev server, run a simple server from the project folder:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Customize

- Swap images by updating the `src` paths in `index.html`.
- Update contact details in the Contact section and in `main.js`.
- Edit the live styles in `theme.css`.
